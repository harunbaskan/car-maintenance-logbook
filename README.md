Car Maintenance Logbook

Final term project for the System Analysis and Design course.

About the Project

This project is a web application where users can manage their vehicles and maintenance records. With JWT authentication, users can manage their own data in an isolated and secure way.

Technologies
Backend: Node.js + Express
Database: MongoDB Atlas (Mongoose ODM)
Authentication: JWT (JSON Web Token)
Validation: Joi
Testing: Jest
API Documentation: Swagger UI
Frontend: Vanilla JavaScript (SPA)
Features
User registration and login (JWT)
User-specific data isolation
Vehicle management (CRUD)
Maintenance records management (CRUD)
Filtering and search
RESTful API
Installation

For detailed installation instructions, please see the backend/README.md file.
